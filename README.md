# EasyTier-Lite
## 简介
<p>EasyTier简化使用的一种UI</p>
<p>适用于无公网IPv4场景</p>
<p>飞牛版本基于CGI模式实现，尽量减少内存使用</p>

## 功能
- 支持多平台
  - FnOS
  - Windows
  - MacOS (未验证)
  - Linux (未验证)
- 支持多配置
- 支持设置开机自启
- 支持EasyTier常见设置项
