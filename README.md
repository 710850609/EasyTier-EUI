## 简介
<p>EasyTier简化使用的一种UI</p>
<p>适用于无公网IPv4场景</p>
<p>飞牛版本基于CGI模式实现，尽量减少内存使用</p>

## 功能
- 支持多平台
  - FnOS (x86_64, aarch64)
  - Windows (x86_64)
  - Linux (x86_64, aarch64, riscv64, 部分验证)
  - MacOS (intel, arm64, 未验证)
- 支持多配置
- 支持设置开机自启
- 支持EasyTier常见设置项
- 提供动态初始节点（数据来自网络社区）
- 支持本地初始节点检测：可用性、延迟、可中转
